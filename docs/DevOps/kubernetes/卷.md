1. 创建cm

创建的语法:
```
kubectl create cm <filename> [--from-file/] <filepath>
```
选项:
- --from-file: 以文件的形式存储cm
```bash
kubectl create cm nginx.conf --from-file /tmp/nginx.conf
```

创建卷:
deployment.yaml :
```yml
spec:
  template:
    spec:
      volumes:
        - name: nginx-conf-volumes
          configMap: nginx-conf
        - name: nginx-ssl-volumes
          secret
```
