解决方案:
假设`kube-apiserver|kube-controller-manager\|kube-scheduler`三者出现了问题
如果是containerd:  
```bash
# 使用 crictl（containerd）
crictl ps -a | grep "kube-apiserver\|kube-controller-manager\|kube-scheduler"
```
