## Windows

1. 设置Kubectl文件的环境变量

```shell
KUBECTL

F:\Kubernetes\kubectl
```

2. 设置远程服务器的 `kubeconfig` 配置文件, `KUBECONFIG`该值固定

```shell
KUBECONFIG

F:\Kubernetes\kubectl\config
```

3. kubectl别名
   powershell7:

```shell
Set-Alias -Name k -Value kubectl
```

Helm:

```shell
HELM

F:\Kubernetes\helm
```
