数据库文件名称最佳实践:
```
{version}_{title}.up.{extension}
{version}_{title}.down.{extension}

```

例如:
```
01_initialize_schema.down.sql
01_initialize_schema.up.sql
```

或适当的时间戳:
```
1500360784_initialize_schema.down.sql
```
