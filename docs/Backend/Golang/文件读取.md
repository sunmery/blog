## 读取上传的文件
### 第一种 file.Open + io.ReadAll()
1. 打开文件
2. 读取文件
```go
file,err := scontent.FormFile("file") // 读取客户端发送的文件, key为file
filedata, err2 := file.Open() // 打开文件
data, err3 := io.ReadAll(filedata) // 读取文件
fmt.Println(string(data)) // 将流转为人类可阅读的string形式
```
