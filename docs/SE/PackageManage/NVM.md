# NVM

#PackageManage #命令

## Node.js Version Manage Node.js 版本管理器

- 查看nvm版本

  ```
  nvm -v
  ```

- 列出所有选项

  ```
  nvm -h 
  ```

- 列出可以安装的版本

  ```
  nvm list available
  ```

- 安装 LSTC 版本

  ```sh
  nvm install lts 
  ```

- 安装最新版

  ```
  nvm install latest
  ```

- 查看已安装的版本

  ```
  nvm ls
  ```

- 使用特定的版本

  ```
  nvm use lts / latest / 指定版本号
  ```

- 卸载版本

  ```
  nvm uninstall lts / latest / 指定版本号
  ```
