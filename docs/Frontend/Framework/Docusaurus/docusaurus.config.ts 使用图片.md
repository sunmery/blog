将图片放在`static/img`目录, 然后再这里只需要使用`img/图片名称`即可, 例如
```ts
 links: [
                {
                    items: [
                        {
                            html: `公安联网备案信息: <img src="img/beian.png" alt="备案图标"/>`
                        }
                    ]
                }
            ]
```
